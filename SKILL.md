# 视觉风格选择

渲染前询问用户希望视频是什么风格。

## 推荐默认

```text
浅色财经 MG
```

## 可选风格

给用户展示简单选项：

```text
1. 浅色财经 MG
2. 知识解释风
3. 新闻评论风
4. 强节奏短视频风
```

## 引导方式

不要讲代码。用中文解释画面感：

- 浅色财经 MG：清爽、专业、卡片、数字、关键词高亮。
- 知识解释风：适合讲概念，结构更慢，层层展开。
- 新闻评论风：适合观点表达，标题和结论更有力量。
- 强节奏短视频风：节奏更快，文字冲击更强。

如果用户不选，使用默认风格。

## 下一步

把用户选择交给 `animation-preset-builder`。
